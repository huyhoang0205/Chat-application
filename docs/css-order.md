## Quy trình viết chuẩn (The Flow)
1. Một quy trình viết CSS khoa học thường chia làm các bước sau:
- Layout lớn: Định hình các khối chính (header, main, sidebar, footer).
- Box Model: Xác định kích thước (width, height), khoảng cách (margin, padding) và khung viền (border) của khối đó.
- Positioning: Sắp xếp vị trí (display: flex, grid, position: absolute,...).
- Typography & Styling: Cuối cùng mới là màu sắc, font chữ, hiệu ứng đổ bóng, bo góc.
## Quy tắc viết bên trong một Selector

|Nhóm|Các thuộc tính tiêu biểu|
|----|-------|
|Vị trí (Positioning)|"display, position, top, z-index, float"|
|Box Model|"width, height, margin, padding, border"|
|Kiểu chữ (Typography)|"font-size, line-height, text-align, color"|
|Trang trí (Visuals)|"background, box-shadow, opacity, transition"|
